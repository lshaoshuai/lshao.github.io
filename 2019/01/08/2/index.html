第一步: 在Github上新建个人网站




  • 创建一个github项目
    前往github官网 ,点击右上角的加号，点击 New repository 创建新的仓库，填写Repository name格式为 username.github.io username是你在github上创建的名称。填写完毕后点击Create repository
